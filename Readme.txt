☕ Le Grand Café du Théâtre
Bienvenue sur le dépôt GitHub du site officiel Le Grand Café du Théâtre. Ce projet représente la vitrine en ligne du café, mettant en avant son ambiance chaleureuse, son menu raffiné et ses horaires d’ouverture.

🚀 Technologies utilisées
HTML5, CSS3, JavaScript : Structure et dynamisme du site
Bootstrap : Design responsive et modernisé
AOS (Animate On Scroll) : Animations élégantes
FontAwesome & Bootstrap Icons : Icônes interactives
GitHub Pages : Hébergement du site

🎨 Fonctionnalités
📍 Page d'accueil avec une présentation du café
🍽️ Menu détaillé des plats et boissons
⏰ Horaires d’ouverture mis à jour
📞 Contact rapide avec un lien WhatsApp direct
📧 Email cliquable pour une prise de contact rapide
📍 Carte interactive pour localiser facilement le café

🌍 Déploiement
Le site est hébergé sur vercel et accessible via Lien: https://le-grand-cafe-du-theatre.vercel.app/.
